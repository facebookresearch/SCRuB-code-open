# Prompts

Prompt templates used in the paper, provided verbatim for reproducibility.

| File | Used in |
|------|---------|
| `rubric.md` | Human annotation study (Table 3); also informs ensemble scoring dimensions |
| `ensemble_scoring_prompt.md` | Panel of Disciplinary Experts — sent to each judge model per (response, perspective) pair |
| `adversarial_transform_prompt.md` | Adversarial framing experiment (Stage 1) — rewrites study prompts into four framing conditions |
| `prompt_generation_bbq.md` | SCRuBEval construction — generates open-ended prompts from BBQ scenarios |
| `prompt_generation_hle.md` | SCRuBEval construction — generates open-ended prompts from HLE questions |
| `prompt_generation_model_spec.md` | SCRuBEval construction — generates open-ended prompts from model specification concepts |
| `prompt_quality_filter.md` | SCRuBEval construction — quality review guide used to filter generated prompts |
